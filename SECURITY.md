# Security Policy

## Scope

This repository contains documentation, resource links, and command references. It does not include executable code or deployed infrastructure.

## Reporting

If you find content that could facilitate harm or a factual error with security implications, contact privately before opening a public issue:

**nervi-zz@protonmail.com**

Include: description of the problem, location in the repository, and potential impact.

## Usage notice

All commands and techniques documented here are for use in authorized environments — your own systems, your organization's systems with permission, or purpose-built lab environments. Nothing here is intended for use against systems you don't own or have explicit authorization to test.
